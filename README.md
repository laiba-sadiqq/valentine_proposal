# ❤️ Valentine Proposal Website

An interactive and visually appealing Valentine-themed website designed to create a memorable proposal experience. This project focuses on creative UI, animations, and user interaction.

---

## 🌟 Features

* Interactive proposal flow 
* Smooth animations and transitions
* Mobile-friendly layout
* Custom styling and themed visuals

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript/ typescript
* react

---

## 🎯 Purpose

This project was built to practice front-end development skills, especially:

* UI/UX design
* DOM manipulation
* Event handling
* Creative web interactions
